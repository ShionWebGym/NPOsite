# できること

・マークダウン形式での記事の投稿
・写真の投稿
・トップページへ新しい順番に記事を表示させる

## やり方
md拡張子のファイルを作る

```
---
title: "お知らせ及び記事投稿"
slug: /post0
date: 2023-04-18T17:00:00
---
```

記事のヘッダーを設定する。
data:の形式は変えるとエラーになります。
